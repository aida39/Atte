## 環境構築

 1. ```docker-compose exec php bash```
 2. ```composer install```
 3. .env.exampleファイルから.envを作成し、環境変数を変更
 4. ```php artisan key:generate```
 5. ```php artisan migrate```
 <!-- 6. ```php artisan db:seed``` -->

## 補足事項
